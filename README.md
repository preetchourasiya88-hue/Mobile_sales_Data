📱 Mobile Sales Dashboard Project 

I created a Mobile Sales Dashboard in Excel to analyze and visualize mobile sales data in an interactive way.
The main goal of this project was to convert raw sales data into meaningful business insights using Excel tools and visualization techniques.

In this dashboard, I worked on:

Data Cleaning & Formatting
Pivot Tables & Pivot Charts
KPI Cards
Interactive Slicers
Sales Performance Analysis

This project helped me understand how businesses track sales performance and make data-driven decisions.

🔹 2. KPI (Key Performance Indicators) Used

The dashboard includes important KPIs to measure business performance such as:

📊 Total Sales
📱 Total Mobile Units Sold
💰 Total Profit / Revenue
🛒 Average Sales Value
🌍 Region-wise Sales Performance
🏆 Top Selling Mobile Brand
📅 Monthly Sales Growth
👥 Customer Purchase Analysis

These KPIs help quickly understand overall business performance.

🔹 3. Graphs & Visuals Used

To make the dashboard interactive and easy to understand, I used different charts and graphs:

📈 Line Chart → Monthly Sales Trend
📊 Bar Chart → Brand-wise Sales Comparison
🍩 Donut/Pie Chart → Category or Region Contribution
📉 Column Chart → Profit & Revenue Analysis
🎯 KPI Cards → Highlight Key Metrics
🎛️ Slicers → Filter data dynamically by month, brand, or region

These visuals help users analyze data quickly and make better decisions.

🔹 4. How This Dashboard Helps

This dashboard provides valuable business insights and helps in:

Identifying top-performing mobile brands
Tracking monthly sales growth
Understanding customer buying patterns
Comparing sales performance across regions
Making faster and data-driven business decisions
Improving sales strategy using visual insight
